First Data for Paid Memberships Pro allows merchants to accept all major credit cards for both one-time and recurring membership payments.

== Description ==

The First Data for Paid Memberships Pro plugin adds First Data as a payment method within your Paid Memberships Pro website. First Data makes accepting credit cards simple. 
Accept all major credit cards including Visa, MasterCard, American Express, Discover, JCB, and Diners Club. The First Data for Paid Memberships Pro plugin 
allows your customers to securely use their credit card to pay for both one-time and recurring memberships.

= Features =

* Supports both one-time and recurring transaction types
* Customers can update or cancel existing membership subscriptions
* Supports all Paid Memberships Pro 1.7+ features
* Uses the PMPro built in checkout so the customer never leaves your website
* AVS and CVC responses are provided to assist with fraud prevention

== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater
* Paid Memberships Pro 1.7 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you donâ€™t need to leave your web browser. To do an automatic install,
follow these directions:

1. Log in to your WordPress dashboard
1. Navigate to the Plugins menu and click Add New
1. Search for "First Data for Paid Memberships Pro" and click "Install Now"
1. Activate "First Data for Paid Memberships Pro" from the Plugins page
1. Complete the configuration by navigating to Memberships -> Payment Settings
1. Select "First Data" from the Payment Gateway dropdown list
1. Enter your API Login and Security Key

= Manual installation =

1. Download and unzip the First Data for Paid Memberships Pro plugin
1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate "First Data for Paid Memberships Pro" from the Plugins page
1. Complete the configuration by navigating to Memberships -> Payment Settings
1. Select "First Data" from the Payment Gateway dropdown list
1. Enter your API Login and Security Key

= How do I test the plugin before going live? =

The plugin has a built-in test mode. Navigate to the configuration page at Memberships -> Payment Settings
and select "Sandbox/Testing" from the Gateway Environment dropdown list and then click the "Save Changes" button. 
The API Login and Security Key fields are not required in sandbox mode and can be left blank.
